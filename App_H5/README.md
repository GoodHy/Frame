# qinjia-app-h5

## node 内存溢出问题

运行 npm run fix-memory-limit
运行 npm run dev
根据提示删除相应文件中不识别的标识

## 浏览器中调试

```
在链接后加参数-----debug=true
```

## Project setup

```
npm/cnpm install
npm run dev  ---  启动开发环境服务
npm run mock --- 启动mock环境服务
npm run build-dev --- 构建开发环境发布包
npm run build-pro --- 构建生产环境发布包
npm run lint --- 修复格式化问题
```
